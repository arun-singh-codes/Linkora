# Linkora
Linkora is a LinkedIn-inspired networking platform where users can connect, share posts, manage professional relationships, and explore additional custom features beyond traditional networking.

## Features

- User Authentication
- User Profiles
- Create & Manage Posts
- Send/Accept Connection Requests
- Notifications
- Real-time Updates
- Custom Networking Features

## Tech Stack

- React.js / Next.js
- Node.js
- Express.js
- MongoDB
- Redux
- Material UI
